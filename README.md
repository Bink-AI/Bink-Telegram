## Getting Started

```bash
# Add the bot on Telegram
@BinkDeFiBot

# Start a conversation
/start

# View available commands
/help
```

## Common Commands

### Trading Commands
```bash
# Get token information
/info <token_symbol>

# Sell tokens
/sell <token_symbol>       # Initiate token selling
/confirm_sell             # Confirm sell transaction
/cancel                   # Cancel current transaction

# Buy tokens  
/buy <token_symbol>       # Initiate token purchase
/confirm_buy             # Confirm buy transaction

# Check token details for selling
/sell_detail <token_symbol>

# View your wallet balance
/balance

# Get market analysis
/analyze <token_symbol>
```

### Account Commands
```bash
# Start the bot
/start

# Get help information
/help

# Connect your wallet
/connect

# View your profile
/profile
```

### Market Information
```bash
# Get current prices
/price <token_symbol>

# View trading pairs
/pairs

# Check market stats
/stats
```
